My Draw - An OpenGl Drawing tool

This is a simple drawing program which can draw line and polygons. Also it has ability to fill bounded regions with simple and check board fill.

color_t class

  • It has Red, Green and Blue value to store color information. Integers can range from 0-255

pen_t class

  • pen has size (int)
  • pen has color (color_t)
  • pen has eraseMode (bool)

object_t class

  • this has a pen (pen_t)
  • it is an abstract class for all objects which are drawn on screen
  • it has function like draw and print which draws inherited objects and prints info in fstream.

point_t class

  • it is inherited from object_t so it contains pen
  • it contains x, y (int) to store location of point to be drawn
  • this is the only object which directly used opengl interface to draw
  • its draw function also updates a bool 2d array containing boundary information like line and polygon

line_t class

  • it is inherited from object_t so it contains pen
  • it contains p1, p2 (point_t) to store end points of line
  • it uses bresenham algorithm for all integers to draw line in all quadrants
  • it uses draw function of point to draw a line

polygon_t class

  • it is inherited from object_t so it contains pen
  • it contains a list of points (list) to store vertices of polygon
  • it uses a loop to take a pair of point which initiate a line drawing using line_t

fill_t class

  • it is inherited from object_t so it contains pen
  • it contains pen1, pen2 (pen_t) to store two colors to facilitate check board fill also
  • it has a point (point_t) to know the start point of flood fill
  • it contains a boolean check to identify type of fill
  • it implements a flood fill algorithm for four neighbors and queue implementation

drawing_t class

  • it contains list of objects (list)
  • the list uses polymorphism concept to contain different class object as they are all inherited from object_t
  • it implements different addObject methods for each of the inherited class of object_t class
  • it has draw function to call all objects draw function and similar for print

canvas_t class

  • it has a current pen and background color (pen_t) to save corresponding objects
  • it has width, height (int) to store dimension info
  • it has drawing (drawing_t) to save the drawing
  • it has twoDArray ( vector >) to store boundary information
  • it clear, draw and save which clears objects list, calls draw function of drawing and calls print function of drawing

my_draw main file

  • it has reshape function to update window size variables to get mouse position
  • it has display function to redraw image or draw only last object as required
  • key board and mouse call back functions
  • Use c to load new canvas - then specify size and background color in terminal
  • Use d to initialize new drawing Use 1 to switch on line mode - which on clicking two points will draw the line and 1 to switch of line mode
  • Use 2 to switch on polygon mode - which on clicking will add points to polygon and 2 to draw polygon and for next polygon switch on polygon mode
  • Use f to switch on fill mode - click to fill the bounded area
  • Use c to change color of line and polygon when in fill mode you can choose check board and simple fill, which then will ask for respective colors Use u to undo last action
  • Use l to load a image Use s to save image to images/saveImage file which should be copied so it is not replaced on next save
  • Use esc to close
